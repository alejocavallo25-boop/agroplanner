# agroplanner
Sistema de gestión agropecuaria completo que cubre tres módulos independientes: Agricultura (lotes, costos, alquileres, insumos y ventas), Tambo (producción lechera, ventas de carne, análisis por litro y rinde de indiferencia) y Ganadería (simulador feedlot de invernada y engorde con análisis en ARS y USD).
