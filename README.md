# 🌾 AgroPlanner

> Plataforma web integral de gestión agropecuaria — en producción con usuarios reales.

---

## ¿Qué es AgroPlanner?

AgroPlanner es una aplicación web desarrollada para gestionar de forma completa una explotación agropecuaria. Cubre tres grandes módulos: **Agricultura**, **Tambo** y **Ganadería**, con herramientas financieras, mapas satelitales y soporte multi-usuario.

Está diseñada para correr desde cualquier dispositivo sin instalación, como una **Progressive Web App (PWA)**.

---

## ✨ Funcionalidades principales

### 🌱 Agricultura
- Mapa satelital interactivo con lotes y cultivos georeferenciados
- Registro de siembras, labores y cosechas
- Cálculo automático de márgenes brutos en **ARS y USD**
- Control de costos e ingresos por cultivo

### 🐄 Tambo & Ganadería
- Registro de rodeo y seguimiento individual por animal
- Simulador **feedlot** con análisis de dietas por fase
- Control de sanidad, reproducción y producción lechera

### ⚙️ Sistema
- Multi-usuario con permisos configurables por módulo
- Instalable como PWA (funciona en celular y PC sin app store)
- Exportación de reportes a PDF
- Dashboard con resumen financiero en tiempo real

## 🚀 Cómo ejecutar el proyecto

```bash
# 1. Cloná el repositorio
git clone https://github.com/alejocavallo25-boop/agroplanner.git

# 2. Configurá tu base de datos MySQL e importá el archivo .sql incluido

# 3. Copiá config.example.php a config.php y completá tus credenciales

# 4. Servilo con Apache o Nginx (XAMPP/Laragon funcionan perfectamente)
```
